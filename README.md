<br>

## 📌 프로젝트 소개

평소 읽은 책들을 분야별로 정리하고 소개하고 싶어서 만든 개인 웹사이트입니다.
자기계발, 부동산, 경제/경영, 심리학 4개 카테고리로 분류하여
각 분야의 입문 추천 책들을 한눈에 볼 수 있도록 구성했습니다.

<br>

## 🛠 기술 스택

| 분야 | 기술 |
|------|------|
| Frontend | HTML, CSS, JavaScript |

<br>

## ✨ 주요 기능

- **분야별 카테고리**: 자기계발 / 돈·부동산 / 경제·경영 / 심리학 4개 분야로 분류
- **책 목록 페이지**: 분야별 입문 추천 책 소개 (총 16권)
- **마우스 인터랙션**: 카테고리 hover/클릭 시 색상 변경으로 UX 향상
- **페이지 이동**: 카테고리 클릭 시 해당 분야 페이지로 이동

<br>

## 📁 프로젝트 구조

```
ReBook/
├── ReBook.html       # 메인 페이지 (책 목록 + 카테고리)
├── ReBook.css        # 전체 스타일
├── ReBook.js         # 카테고리 인터랙션 기능
├── 자기계발.html      # 자기계발 분야 책 목록
├── 부동산.html        # 돈/부동산 분야 책 목록
├── 경제경영.html      # 경제/경영 분야 책 목록
└── 심리학.html        # 심리학 분야 책 목록
```

<br>

## 📖 소개 책 목록

| 분야 | 책 제목 |
|------|--------|
| 자기계발 | 역행자, 세이노의 가르침, 슈퍼노멀, 미라클모닝 |
| 돈/부동산 | 서울 자가에 대기업 다니는 김부장 이야기, 나의 돈 많은 고등학교 친구, 부의 추월차선, 부자 아빠 가난한 아빠 |
| 경제/경영 | 주린이가 가장 알고 싶은 최다질문 TOP77, 부자의 그릇, 1%를 읽는 힘, 레버리지 |
| 심리학 | 공감하는 유전자, 설득의 심리학, 인간관계론, 시크릿 |
